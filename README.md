# VyapaarBot

An Agentic AI-powered growth assistant for Bharat's women entrepreneurs and online resellers.

## Problem

Millions of micro-entrepreneurs manage inventory, pricing, customer engagement, and support manually without access to business intelligence tools.

## Solution

VyapaarBot provides an AI-powered business team through specialized agents:
- Trend Agent
- Marketing Agent
- Price Agent
- Query Agent

All recommendations are delivered through a daily Morning Business Brief.

## Features

- Trend Analysis
- Marketing Message Generation
- Pricing Recommendations
- Customer Query Assistant
- Morning Business Brief Dashboard

## Tech Stack

- HTML/CSS/JavaScript
- Python (Planned)
- Gemini API (Planned)
- Streamlit (Planned)
